Neste projeto, utilizei habilidades avançadas de Programação em Python e manipulação de dados para desenvolver um painel gerencial inovador para uma empresa marketplace de delivery de comida. Integrando pensamento estratégico e lógica de negócios, juntamente com ferramentas de desenvolvimento web como Streamlit e Github, o resultado final foi um painel interativo com as principais métricas de negócio e vários Insights.
Hospedado em um ambiente Cloud, o painel é acessível através de um link web em qualquer dispositivo conectado à internet.
